    Setup - Guide for Windows

    1. Terminal-可在终端(Power Shell/Git Bash)输入命令与计算机进行交互(bash命令 cd/ls)

    2. Python-打开Python(~~python3~~/py/python),使用exit()/Ctrl-Z并回车返回命令行

    3. Text Editor-推荐使用VSCode;Others:Atom、Vim、Emacs、Pycharm、Sublime Text

    4. VSCode

      • UNIX指令-

        • mkdir ~/Desktop/example 新建文件夹
        • cd ~/Desktop/example 跳转至指定文件夹
        • code . 还可以通过命令行在当前工作目录下打开VS CODE

      • Editing files-Welcome.py

      python **def** **welcome**(name): print('Welcome to CS61A, %s!' % name)

      • Running Python

        python3 -i welcome.py

        • -i 为以交互模式启动
        • >>> 意为准备好接受命令

        welcome('MDKIKOU')

      • 键盘快捷键

        • Ctrl + ` 打开集成终端
        • Ctrl + D 突出显示当前单词
        • Ctrl + (Shift) + Tab 移动到(上)下一个选项卡
        • Ctrl + v 复制到光标上方行

      • Extension

        • Ctrl + Shift + S 快捷打开

    5. Walkthrough:Using The Terminal

      • CLI(command line interface) & GUI(graphical user interface)
      • Terminal / python interpreter / code editor

    6. Walkthrough:Organizing Files

      • ls lists all the files and folders in the current directory
      • cd change directory
      • cd ~ change directory to home directory
      • mkdir CS61A makes a new directory

      • Primitive Expressions
      • Arithmetic Expressions
        • ** 幂运算符
        • / Floating point division 浮点除法（将第一个数字除以第二个数字，即使数字相等，也计算为具有小数点的数字）
        • // Floor division 楼层除法（将第一个数字除以第二个数字，然后向下舍入，计算结果为整数）
        • % Modulo 模（计算除法遗留正余数）
        • 计算顺序：PEMDAS (Parentheses, Exponentiation, Multiplication / Division, Addition / Subtraction)
